# gh-workflow

A project to explore and demonstrate GitHub Actions workflows.

## Live Demo

View the project live at: [gh-workflow GitHub Pages](https://vishakhamangtani23.github.io/gh-workflow/)

## Features
- Automated CI/CD with GitHub Actions
- Deployment to GitHub Pages
- Example workflows for testing and deployment

## Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/vishakhamangtani23/gh-workflow.git
   ```
2. **Explore workflows:**
   - Check `.github/workflows/` for sample YAML files.
3. **View the site:**
   - Open [https://vishakhamangtani23.github.io/gh-workflow/](https://vishakhamangtani23.github.io/gh-workflow/) in your browser.
